# School Management System

A comprehensive school management system with features for attendance tracking, HR management, fee payments, and more.

## Project Structure

- `APP/` - Frontend application (React/Vite)
- `backend/` - Backend API server (Node.js/Express)

## Setup

### Backend
```bash
cd backend
npm install
npm start
```

### Frontend
```bash
cd APP
npm install
npm run dev
```

## Environment Variables

Copy the `.env.example` files and configure your environment:
- `backend/.env.production.template` → `backend/.env`
- `APP/.env.example` → `APP/.env`

## Features

- Student & Staff Management
- Attendance Tracking (Manual & Biometric)
- HR & Salary Management
- Fee Payment & Invoice System
- Guardian Portal
- Report Cards
- Budget & Expense Management
- Ethiopian Calendar Support
- Mobile & Desktop Apps
