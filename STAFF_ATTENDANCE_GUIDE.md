# Staff Attendance System - User Guide

## Overview
The Staff Profile attendance tab provides a quick overview of student attendance and easy access to the full attendance management system.

## How to Access
1. Login as a staff member (teacher/class teacher)
2. Navigate to your Staff Profile
3. Click on the **Attendance** tab in the bottom navigation

## Understanding the Attendance Tab

### 1. Quick Statistics Cards
At the top, you'll see 4 colorful cards showing:
- **Total Students** - Number of students in your class
- **Present** - Students marked as present today
- **Late** - Students who arrived late
- **Absent** - Students who are absent

### 2. Manage Attendance Button
The purple **"Manage Attendance"** button takes you to the full attendance system where you can:
- Mark daily attendance
- View weekly attendance records
- Edit past attendance
- Generate attendance reports
- Configure attendance settings

### 3. Students List
Shows all students in your assigned class with:
- **Student Avatar** - Profile picture or initials
- **Student Name** - Full name
- **Student ID** - Class ID number
- **Machine ID** - Biometric device ID (if available)
- **Attendance Status Badge**:
  - 🟢 **Present** - Green badge
  - 🟡 **Late** - Yellow/Orange badge
  - 🔴 **Absent** - Red badge
  - ⚪ **Not Marked** - Gray badge (attendance not yet recorded)

### 4. School Days Information
At the bottom, you'll see:
- Active school days (Monday-Friday by default)
- Note about weekly schedule

## How to Mark Attendance

### Step 1: Access Full Attendance System
Click the **"Manage Attendance"** button

### Step 2: Select Week
- Choose the current week or any past week
- System shows Ethiopian calendar dates
- Weeks start from Monday

### Step 3: Mark Students
For each student, you can mark:
- ✅ **Present** - Student is in class
- ⏰ **Late** - Student arrived after check-in time
- ❌ **Absent** - Student is not present
- 📋 **Leave** - Student has approved leave

### Step 4: Save
Click the **Save** button to record attendance

## Features

### Auto-Absent Marking
- System automatically marks students as absent if they don't check in by the configured time
- Can be enabled/disabled in settings

### Machine Integration
- Supports biometric attendance machines
- Students with Machine IDs can check in automatically
- Manual override available for teachers

### Dual Shift Support
- Shift 1: Morning classes (default 7:00 AM - 9:00 AM)
- Shift 2: Afternoon classes (default 1:00 PM - 3:00 PM)
- Each shift has separate check-in times

### Attendance Reports
- Weekly summaries
- Monthly reports
- Individual student attendance history
- Export to Excel/PDF

## Settings Configuration

### Time Settings
Navigate to: **Academic → Student Attendance Time Settings**

Configure:
1. **Check-in Times**
   - Start time (when students can check in)
   - End time (last time for on-time check-in)
   - Late threshold (after this time = late)
   - Absent marking time (auto-mark absent)

2. **Shift Assignments**
   - Assign classes to Shift 1 or Shift 2
   - Different times for each shift

3. **School Days**
   - Select active school days
   - Weeks show only selected days

4. **Auto-Absent**
   - Enable/disable automatic absent marking
   - Runs at configured absent marking time

## Troubleshooting

### Students Show "Not Marked"
- Attendance hasn't been recorded yet for today
- Click "Manage Attendance" to mark attendance
- Or wait for auto-absent marking if enabled

### Can't See Attendance Tab
- Only class teachers see the attendance tab
- Contact admin to assign you as a class teacher
- Check: **Academic → Class Teacher Assignment**

### Wrong Class Showing
- System shows your first assigned class
- If you teach multiple classes, use the full attendance system
- Select different class from dropdown

### Attendance Not Saving
- Check internet connection
- Verify you have permission to mark attendance
- Try refreshing the page (Ctrl+Shift+R)
- Contact system administrator
